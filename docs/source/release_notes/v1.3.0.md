# 1.3.0

## What's changed
The major change in this release is a new **dark mode** and other theming support!

### New
* Upgrade theming by @ThomasKroes in https://github.com/ManiVaultStudio/core/pull/817
* Update Qt, CI runners and cpp version by @alxvth in https://github.com/ManiVaultStudio/core/pull/813, https://github.com/ManiVaultStudio/core/pull/835, 
   * Update Qt to `6.8.2` from `6.3.2`
   * Update [ads](https://github.com/githubuser0xFFFF/Qt-Advanced-Docking-System/releases/tag/4.4.0) to `4.4.0` from `4.3.1`
   * Update [biovault_bfloat16](https://github.com/biovault/biovault_bfloat16/) to `1.0.2` from `1.0.1`
   * Update [quazip](https://github.com/stachenov/quazip) to `1.5` from `1.4` 
   * Update C++ version to `20` from `17`
   * Update [CMake](https://cmake.org/cmake/help/latest/release/3.22.html) minimum required version to `3.22` from `3.17`
* Add theme icon by @ThomasKroes in https://github.com/ManiVaultStudio/core/pull/806
