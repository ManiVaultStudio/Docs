Learning Center
===============

The **Learning Center** provides **contextual, plugin-specific guidance** while working with ManiVault.

It surfaces **just-in-time information**—such as interaction hints and shortcut summaries—based on the currently available plugins and views.

![Learning center](../../assets/learning_center.gif)

*Plugin-specific onboarding content in the Learning Center.*

---

## Overview

- Supports **onboarding** and feature **discoverability**
- Content is **defined by plugins** and registered via their factories
- Presents concise guidance and **shortcut summaries**
- Integrated and rendered by the core
- Non-intrusive and optional

The Learning Center helps users quickly understand *what a plugin does* and *how to interact with it*.
